# 文例メーカー（BunreiMaker）事業計画書

## 1. 事業概要

### サービス名
文例メーカー（BunreiMaker）

### サービス概要
ビジネスシーンで必要となる各種文章（お詫びメール、お礼メール、退職届、挨拶文など）のテンプレートを提供し、ユーザーが必要事項を入力するだけで適切な文章を自動生成する無料ウェブツール。

### 収益モデル
Google AdSense による広告収益

### 市場の特徴
- ビジネス文章関連キーワードはCPCが高く、AdSense RPMが$5〜$10を見込める
- 季節変動があるキーワード（年賀状：10〜12月、退職届：2〜3月）が含まれ、年間を通じた安定したトラフィックが期待できる
- 競合は多いが、「生成ツール」としてのインタラクティブな機能を持つサイトは少ない
- スマートフォンからの検索が多く、モバイル対応は必須

## 3. 競合分析

### 主な競合
1. **文例サイト（静的コンテンツ型）**：例文を羅列する形式。ユーザーが自分で編集する必要がある。
2. **ビジネスマナーサイト**：マナー解説が主体で、テンプレートは補足的。
3. **テンプレートダウンロードサイト**：Word/PDF形式でのダウンロードが中心。

### 差別化ポイント
- リアルタイムプレビュー機能（入力と同時に文章が変化）
- ワンクリックコピー機能
- 複数テンプレートの切り替え
- トーン（フォーマル/カジュアル）の調整機能
- 完全クライアントサイド処理（プライバシーの確保）
- Apple風のモダンなUI/UX

## 4. 収益予測

### 前提条件
- AdSense RPM（ビジネス系キーワード）：$5〜$10（約750〜1,500円/1,000PV）
- 検索エンジンからの月間PV目標

### フェーズ別目標

#### フェーズ1（1〜3ヶ月目）：立ち上げ期
- 月間PV：5,000〜10,000
- 月間収益：5,000〜15,000円
- 施策：全ページ公開、Google Search Console登録、基本的なSEO対策

#### フェーズ2（4〜6ヶ月目）：成長期
- 月間PV：30,000〜50,000
- 月間収益：30,000〜75,000円
- 施策：SEOコンテンツの拡充、内部リンク最適化、SNSでの認知拡大

#### フェーズ3（7〜12ヶ月目）：拡大期
- 月間PV：80,000〜150,000
- 月間収益：80,000〜150,000円
- 施策：新規ジェネレーター追加（契約書、請求書、領収書等）、被リンク獲得

#### フェーズ4（13ヶ月目〜）：安定期
- 月間PV：150,000〜250,000
- 月間収益：150,000〜200,000円以上
- 施策：既存コンテンツの改善、ユーザーフィードバックに基づく機能追加

### 月間200,000円達成に必要な条件
- RPM ¥1,000の場合：月間200,000PVが必要
- RPM ¥1,500の場合：月間133,000PVが必要

ビジネスキーワードのRPMは一般キーワードより高い傾向にあるため、RPM ¥1,000〜¥1,500は現実的な数値。

## 5. SEO戦略

### オンページSEO
- 各ジェネレーターページにSEO解説コンテンツを配置（1,000文字以上）
- 構造化データ（JSON-LD）の実装（HowTo, FAQPage）
- 内部リンクの最適化
- Core Web Vitals の最適化（クライアントサイドのみの軽量設計）

### コンテンツ戦略
- 既存10カテゴリの深堀り（テンプレート追加、解説拡充）
- 新規カテゴリの追加（四半期ごとに2〜3カテゴリ）
  - 候補：見積書送付メール、請求書送付メール、議事録テンプレート、報告書テンプレート
- 季節性コンテンツの事前準備（年賀状：9月、退職届：1月）

### テクニカルSEO
- sitemap.xml の定期更新
- モバイルファースト設計
- ページ表示速度の最適化（外部リソース最小化）
- canonical タグの設定

## 6. 運営コスト

### 初期費用
- ドメイン取得：約1,500円/年
- サーバー（静的ホスティング）：0円（GitHub Pages / Cloudflare Pages等）

### 月間運営費
- サーバー代：0円（静的サイトのため無料ホスティング可能）
- ドメイン更新：約125円/月（年間1,500円）

**実質的な運営コストはほぼゼロ。** クライアントサイドのみで動作する静的サイトのため、サーバー費用が不要。

## 7. リスクと対策

| リスク | 対策 |
|---|---|
| 検索順位の変動 | 複数キーワードでの分散、コンテンツの質向上 |
| AdSense RPMの低下 | コンテンツの質による訪問者の滞在時間向上 |
| 競合の参入 | UI/UXの差別化、機能の継続的改善 |
| Googleポリシー変更 | ポリシーの定期確認、代替広告ネットワークの検討 |
