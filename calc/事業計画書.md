# CalcBox（カルクボックス）事業計画書

## 1. サイトコンセプト

### サイト名
CalcBox（カルクボックス）

### タグライン
「お金の計算を、もっとカンタンに。」

### コンセプト
暮らしに関わるお金の計算（手取り、住宅ローン、税金、年金等）を、正確かつわかりやすく提供する無料オンライン計算ツールサイト。Apple風のミニマルで信頼感のあるデザインで、ユーザーに「このサイトの計算結果なら信頼できる」と思わせるブランドを構築する。

### ターゲットユーザー
| セグメント | 年齢層 | ニーズ |
|---|---|---|
| 会社員・公務員 | 25-55歳 | 手取り額の把握、住宅ローン検討、ふるさと納税上限額 |
| 転職検討者 | 25-40歳 | 年収別の手取り比較 |
| 住宅購入検討者 | 30-45歳 | 住宅ローンシミュレーション、賃貸vs持ち家比較 |
| 退職予定者 | 50-65歳 | 退職金の税金計算、年金受給額シミュレーション |
| 投資初心者 | 20-40歳 | 複利計算、つみたてNISAシミュレーション |
| フリーランス・自営業 | 25-55歳 | 国民健康保険料、所得税計算 |

## 2. 収益モデル

### メイン収益源: Google AdSense
金融・税金・不動産関連のキーワードはAdSenseのRPM（Revenue Per Mille）が高い。

| カテゴリ | 想定RPM | 理由 |
|---|---|---|
| 住宅ローン計算 | $10-$15 | 住宅ローン広告の入札単価が高い |
| ふるさと納税 | $8-$12 | ふるさと納税サイトの広告費が高い |
| 手取り計算 | $6-$10 | 転職サイト・保険系の広告 |
| 所得税・住民税 | $8-$12 | 税理士・会計ソフト系の広告 |
| 退職金計算 | $8-$12 | 資産運用・保険系の広告 |
| 年金シミュレーション | $6-$10 | 保険・投資系の広告 |
| 複利計算 | $8-$12 | 証券会社・投資信託の広告 |
| 賃貸vs持ち家 | $10-$15 | 不動産・住宅ローン広告 |
| 国民健康保険 | $5-$8 | 保険系広告 |
| BMI計算 | $3-$5 | 健康食品・ダイエット系 |

**加重平均RPM: 約$8-$10（1,000-1,300円/千PV）**

## 5. 競合分析

### 主要競合サイト

| サイト名 | 強み | 弱み | 推定月間PV |
|---|---|---|---|
| keisan.casio.jp | CASIO運営の信頼性、豊富な計算ツール | デザインが古い、UIが複雑 | 500万+ |
| zeiri4.com（税理士ドットコム） | 税務の専門性、税理士監修 | 計算ツール特化ではない | 300万+ |
| juusetsu-navi.jp | 住宅ローン特化 | 住宅ローンのみ | 50万+ |
| calc-tools.com | 幅広い計算ツール | デザインが古い、広告過多 | 100万+ |
| money-forward.com | ブランド力、総合力 | 計算ツールは副次的 | 1000万+ |

### CalcBoxの差別化ポイント
1. **デザイン品質**: Apple風のクリーンで信頼感のあるデザイン（競合は古いUI/UXが多い）
2. **UX重視**: リアルタイム計算、結果の視覚化（チャート・グラフ）、モバイル最適化
3. **SEO特化**: 金融KWに特化したコンテンツ構造、構造化データ対応
4. **表示速度**: 静的HTML + クライアントサイドJS で高速（サーバーサイド不要）
5. **広告の適切な配置**: 過度な広告を避け、ユーザー体験を維持

## 6. SEO戦略

### テクニカルSEO
- 静的HTML（サーバー不要、高速表示）
- 適切なmeta description、canonical URL
- sitemap.xml、robots.txt
- モバイルフレンドリー対応
- Core Web Vitals最適化（LCP < 2.5s、CLS < 0.1）

### コンテンツSEO
- 各計算ツールページに1,000-2,000文字のSEO解説コンテンツ
- H1, H2, H3の適切な見出し構造
- FAQ構造化データ（将来追加）
- 内部リンクの最適化（関連ツールへの誘導）

### E-E-A-T対策
- 計算ロジックの根拠（法令・税率）を明記
- 「概算値である」旨の免責表記
- 運営者情報、プライバシーポリシー、利用規約の整備
- 定期的なコンテンツ更新（税制改正時）
