# CheckNote（チェックノート）事業計画書

## 1. サービス概要

- **サイト名**: CheckNote（チェックノート）
- **URL**: https://checknote.jp
- **コンセプト**: 人生のライフイベントに必要な手続き・準備をチェックリスト形式で管理できる無料Webサービス
- **ターゲット**: 引越し・結婚・転職・確定申告などを控えた20〜40代の日本人

## 4. SEO戦略

### 4-1. コンテンツSEO

- 各チェックリストページに1,500〜3,000文字のSEOコンテンツセクションを設置
- FAQ構造化データの実装
- 関連キーワードのロングテール対策
- 季節性を意識した更新（確定申告は1〜3月、引越しは2〜4月がピーク）

### 4-2. テクニカルSEO

- セマンティックHTML5構造
- Core Web Vitals最適化（LCP < 2.5s, FID < 100ms, CLS < 0.1）
- 構造化データ（HowTo, FAQPage）の実装
- モバイルファースト設計
- サイトマップ・robots.txtの設置

### 4-3. 内部リンク戦略

- トップページから各チェックリストへの導線
- 関連チェックリスト同士の相互リンク
- パンくずリストの実装

## 6. 競合分析

| 競合 | 強み | 弱み |
|---|---|---|
| 引越し侍 | ドメインパワー、引越し業者DB | チェックリスト機能なし |
| ゼクシィ | ブランド力、結婚情報の網羅性 | 登録が必要、UIが複雑 |
| リクナビNEXT | 求人数、ブランド力 | チェックリスト機能なし |
| Notion テンプレート | カスタマイズ性 | アカウント必要、日本語情報不足 |

**CheckNoteの差別化ポイント**:
- 登録不要・無料・シンプル
- 複数ライフイベントを一元カバー
- Apple風のクリーンなUI/UX
- localStorage保存で手軽

## 7. 技術スタック

- **フロントエンド**: HTML5 + CSS3 + Vanilla JavaScript
- **ホスティング**: Cloudflare Pages（無料枠）
- **ドメイン**: checknote.jp（年額約1,500円）
- **フォント**: Google Fonts（Inter + Noto Sans JP）
- **アクセス解析**: Google Analytics 4
- **広告**: Google AdSense
- **アフィリエイト**: A8.net, もしもアフィリエイト, バリューコマース
