# IconJP（アイコンJP）事業計画書

## 1. 事業概要

### サービス名
IconJP（アイコンJP）

### コンセプト
日本語UIに最適化された、無料で使えるSVGアイコンライブラリ。
日本の文化・ビジネスシーンに特化したアイコンを提供し、国内のWeb制作者・デザイナーをターゲットとする。

### URL
https://iconjp.com

---

## 2. ターゲットユーザー

- 個人ブロガー・アフィリエイター

## 6. 競合分析

| サービス | 強み | 弱み |
|----------|------|------|
| Feather Icons | 洗練されたデザイン | 日本特有アイコンなし |
| Lucide | 豊富な種類 | 日本語対応なし |
| ICOOON MONO | 日本語対応 | SVG最適化が弱い、デザインが古い |
| IconJP（当サービス） | 日本語UI特化 + モダンデザイン | 後発・知名度不足 |

### 差別化ポイント
1. 日本文化・ビジネスに特化したアイコン（印鑑、名刺、おにぎり、桜など）
2. 日本語・英語の両方で検索可能
3. カスタマイズ機能（色・サイズ変更）をブラウザ上で提供
4. MITライセンスで商用利用可能
