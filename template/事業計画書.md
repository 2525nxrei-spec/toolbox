# テンプレBOX 事業計画書

## 1. 事業概要

### サービス名
テンプレBOX（TemplateBOX）

### サービス概要
日本のビジネスシーンで必要な文書テンプレート（履歴書、職務経歴書、請求書、見積書、契約書等）を無料で提供するウェブサイト。HTML形式のテンプレートにより、ブラウザから直接編集・印刷・PDF保存が可能。

### 収益モデル
Google AdSense による広告収入（ディスプレイ広告）

| 報告書 テンプレート | 14,800 | 低 | 8-12% |

| 秘密保持契約書 テンプレート | 8,100 | 低 | 8-12% |

**合計月間検索ボリューム：約329,300回**

### 競合分析
- 既存大手（bizocean、テンプレートBANK等）はWord/Excel形式が中心
- 会員登録が必要なサイトが多く、ユーザーの離脱要因となっている
- デザインが古いサイトが多い

### 差別化ポイント
1. **登録不要**：メールアドレスの入力すら不要
2. **HTML形式**：特別なソフト不要でブラウザから直接利用可能
3. **モダンデザイン**：Apple風のクリーンなUI/UX
4. **SEO最適化**：各テンプレートページに使い方ガイド・SEOコンテンツ充実

## 3. 収益予測

### 前提条件
- Google AdSense RPM（1,000PVあたり収益）：300-500円（ビジネス系サイト平均）
- 平均ページ/セッション：2.5ページ
- 広告ユニット：各ページ3-4枠

### フェーズ別収益予測

#### Phase 1：立ち上げ期（1-3ヶ月目）
- 月間PV：5,000-15,000
- ロングテールキーワードからの流入開始
- **月間収益：1,500-7,500円**

#### Phase 2：成長期（4-6ヶ月目）
- 月間PV：30,000-80,000
- 主要キーワードでの順位上昇
- コンテンツ追加（テンプレート追加・ブログ記事）
- **月間収益：9,000-40,000円**

#### Phase 3：拡大期（7-12ヶ月目）
- 月間PV：100,000-300,000
- 主要キーワードで上位表示を実現
- SNSシェアによる自然流入増加
- **月間収益：30,000-150,000円**

#### Phase 4：安定期（13-18ヶ月目）
- 月間PV：400,000-600,000
- ブランド認知の確立
- リピーター増加
- テンプレート数：30-50種類に拡大
- **月間収益：120,000-300,000円**

### 月間200,000円達成の条件
- 必要月間PV：約500,000PV（RPM 400円の場合）
- 必要オーガニック流入：約200,000セッション/月
- 主要キーワード3-5個で検索結果1ページ目表示が必要
- **達成見込み時期：運用開始後12-15ヶ月目**

## 4. SEO戦略

### オンページSEO
- 各テンプレートページに2,000-3,000文字のSEOコンテンツ
- 構造化データ（JSON-LD）の実装
- 内部リンク最適化（関連テンプレート相互リンク）
- Core Web Vitals最適化（静的HTML、軽量CSS）

### コンテンツ戦略
1. テンプレートページ（10ページ → 30ページに順次拡大）
2. ブログ記事（書き方ガイド、ビジネスマナー記事）
3. カテゴリページの充実

### テクニカルSEO
- サイトマップXML
- robots.txt
- モバイルフレンドリー（レスポンシブ対応）
- ページ速度最適化（外部依存最小限）
- SSL対応

### リンク構築
- SNS公開（Twitter/X、はてなブックマーク）
- ビジネス系メディアへの寄稿
- 無料ツール系まとめサイトへの掲載依頼

## 9. まとめ

テンプレBOXは、最小限の初期投資（年間約3,000円）で開始可能な、高い収益ポテンシャルを持つウェブサービスです。月間329,300回の検索需要がある市場において、登録不要・無料・モダンデザインという差別化要因により、運用開始後12-15ヶ月で月間200,000円の収益目標達成を見込んでいます。

静的サイトによる運用コストの最小化と、テンプレートの段階的拡充により、安定的かつスケーラブルな収益基盤の構築が可能です。
