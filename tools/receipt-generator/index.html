領収書作成

個人事業主・フリーランス向け領収書作成ツール。フォーム入力で即作成・印刷。収入印紙の要否も自動判定。

このツールを使わない場合の損失
✕領収書の発行漏れ → 取引先の経費処理が滞り信用低下
✕印紙税の貼付忘れ → 過怠税（本来の3倍）のリスク
✕記載不備 → インボイス制度で仕入税額控除が認められない
ToolBoxなら → 無料・印紙税自動判定・インボイス制度対応の領収書を即作成

領収書作成ツールの使い方

左側のフォームに発行日・宛名・金額・但し書きなどを入力すると、右側にリアルタイムで領収書のプレビューが表示されます。完成したら「印刷 / PDF保存」ボタンで印刷・PDF化できます。

収入印紙の自動判定

金額を入力すると、印紙税法に基づいて収入印紙の要否と金額を自動判定します。5万円未満は非課税、5万円以上は金額に応じた印紙税額を表示します。

領収書の法的要件

民法第486条では、弁済をした者は受取証書（領収書）の交付を請求できると定められています。領収書は取引の証拠書類として重要な役割を果たします。

  • 民法第486条（受取証書の交付請求権） — 弁済をした者は、弁済を受領した者に対して受取証書の交付を請求できます。
  • 印紙税法（収入印紙の貼付義務） — 売上代金に係る金銭の受取書は、記載金額に応じた印紙税の納付が必要です。

印紙税額の目安:

  • 5万円未満: 非課税
  • 5万円以上100万円以下: 200円
  • 100万円超200万円以下: 400円

インボイス制度における適格簡易請求書としての要件

適格請求書発行事業者が発行する領収書は、適格簡易請求書（簡易インボイス）として認められます。以下の記載要件を満たす必要があります。

  • 適格請求書発行事業者の氏名・名称および登録番号
  • 取引年月日
  • 取引の内容（軽減税率対象の場合はその旨）
  • 税率ごとに区分した合計額（税込または税抜）

※ 本ツールは領収書の記載項目を網羅していますが、個別の税務判断については税理士にご相談ください。法的効力を保証するものではありません。

個人事業主の領収書活用ケース

  • フリーランスのWeb制作・デザイン — クライアントから制作費を受領した際の領収書発行。源泉徴収後の金額を正確に記載。
  • コンサルティング・講師業 — セミナー講演料やコンサルティング報酬の受領時に発行。但し書きに具体的な業務内容を記載。
  • 個人事業の物販・サービス提供 — 商品販売やサービス提供の代金受領時の正式な領収書を即作成。
  • 副業・個人間取引 — 副業の報酬や個人間取引の支払い証明として。確定申告の際の証憑書類に活用。
